## Description

ResumeIQ is a full-stack AI-powered web application designed to analyze resumes and generate meaningful insights to help users improve their professional profiles.

The platform leverages **Google Gemini AI** to perform intelligent resume evaluation, extract technical and professional skills, and provide personalized improvement recommendations.

In addition to AI-driven analysis, ResumeIQ includes secure authentication features such as **email verification**, **password recovery via Brevo**, and **job recommendations powered by Adzuna API**.

# Running the Project Locally

## 1. Clone Repository

```bash
git clone https://github.com/your-username/resumeiq.git
cd resumeiq
```

---

## 2. Open Project in IDE

Open the project using:

* IntelliJ IDEA
* Eclipse

After opening:

* Load `pom.xml`
* Allow Maven to install project dependencies

---

## 3. Configure Application Credentials

Update the `application.properties` file.

### Database Configuration (MySQL)

```properties
spring.datasource.url=YOUR_DB_URL
spring.datasource.username=YOUR_DB_USERNAME
spring.datasource.password=YOUR_DB_PASSWORD
```

### Google OAuth (Sign-In)

```properties
spring.security.oauth2.client.registration.google.client-id=YOUR_GCP_CLIENT_ID
spring.security.oauth2.client.registration.google.client-secret=YOUR_GCP_CLIENT_SECRET
```

### Google Gemini AI

```properties
genKey=YOUR_GEMINI_API_KEY
```

### Email Service (Brevo)

```properties
apiKey=YOUR_BREVO_API_KEY
```

### Job Recommendation API (Adzuna)

```properties
application-id=YOUR_ADZUNA_APP_ID
application-api-key=YOUR_ADZUNA_API_KEY
```

### JWT Secret

```properties
jwt-key=YOUR_SECRET_KEY
```

Generate a secure secret key before deployment.

---

## 4. Run Backend

Launch:

```plaintext
ResumeAnalyserApplication.java
```

or

```bash
mvn spring-boot:run
```

---

## 5. Access Application

```plaintext
http://localhost:8080
```

# Important Notes

### AI Configuration

ResumeIQ currently supports **Google Gemini AI**.

To switch providers, update the AI implementation inside:

```plaintext
appservice.java
```

---

### Mail Configuration

Email functionality currently uses **Brevo API**.

To use another provider, update:

```plaintext
mailservice.java
```

---

### Model Maintenance

AI models may evolve over time.

If the configured Gemini model becomes deprecated, update the model configuration inside:

```plaintext
appservice.java
```

# Frontend Development

Do not modify files directly inside the backend `static` directory.

## Run Frontend Independently

```bash
cd frontend-src
npm install
npm run dev
```

Starts the React development environment.

---

## Build Frontend for Deployment

```bash
cd frontend-src
npm run build
```

---

## Backend Static Directory Structure

```plaintext
static/
├── assets/
│   ├── *.css
│   ├── *.js
├── index.html
```

Deployment Steps:

1. Remove old `index.html`
2. Delete previous `assets`
3. Copy files from `dist`
4. Paste into backend `static`
